About the Project

This web application helps predict the productivity of garment workers based on input parameters like targeted productivity, SMV, overtime, idle time, incentives, and more.

The model was trained using historical production data and uses machine learning (XGBoost) to make predictions. The goal is to assist management in estimating workforce productivity and making informed decisions in garment manufacturing.

Built using Python, Flask, HTML, and CSS.
